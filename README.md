# JEDA Design Thinking Agent Studio

> **Subjudul**: Agentic Design Thinking Workspace for Pinjol, Paylater, Judi Online, and Debt-Stress Intervention in Indonesia
>
> **Konteks Proyek**: Dibuat khusus untuk **PIDI Hackathon x Digdaya 2026**
>
> **Kategori**: *Problem Statement 1 - Penguatan Ketahanan dan Inovasi Keuangan (Fokus: Resiliensi Keuangan Ritel & Perlindungan Konsumen)*

---

## 1. Deskripsi Produk

**JEDA (Jaga Emosi dan Dana)** adalah platform resiliensi keuangan retail dan intervensi perilaku hulu (*behavioral finance*). JEDA dirancang untuk melindungi masyarakat Indonesia dari jebakan kredit konsumtif impulsif (*pinjol/paylater*) serta adiksi judi online. JEDA membantu pengguna menstabilkan emosi krisis keuangan, memetakan kewajiban utang secara visual berdasarkan tingkat keamanan hukum, dan mendampingi transisi mereka hingga siap menjadi calon investor ritel yang mandiri, rasional, dan sadar risiko.

JEDA bertindak sebagai **saringan hulu** yang mempersiapkan pengguna sebelum memasuki pasar modal formal. JEDA berkeyakinan bahwa pengguna yang masih terjerat utang konsumtif berbunga tinggi (10-30% per bulan) tidak rasional dipaksa membeli reksa dana atau saham, karena beban bunga utang jauh melampaui imbal hasil investasi mana pun. JEDA memulihkan kesehatan arus kas (*cashflow*) terlebih dahulu sebelum mendampingi edukasi investasi dasar.

## 2. Fitur Utama Platform

1. **Dashboard & Peta Strategis**: Menyajikan konseptualisasi JEDA, akar masalah yang diselesaikan, relevansi terhadap perluasan investor ritel baru, dan batasan etis perlindungan konsumen.
2. **Simulasi Pengguna JEDA (Interactive Self-Check)**: Memandu demo perjalanan mandiri pengguna melalui asesmen tingkat stres, pengelompokan tingkat risiko (Rendah hingga Kritis), dan perumusan rencana pemulihan 7-hari serta stabilisasi 30-hari.
3. **Debt-to-Investor Readiness Pathway**: Peta jalan interaktif 8 tahapan transisi keuangan hulu, dilengkapi checklist perilaku mandiri yang menghasilkan *Investor Readiness Score* serta rekomendasi edukatif.
4. **Survey Insight Dashboard**: Visualisasi data survei (simulasi 450 responden mock) mengenai titik pusing pengguna (*pain points*), pemicu impulsif, dan pemetaan langsung dari kebutuhan pasar ke solusi desain JEDA.
5. **Double Diamond Workflow Board**: Menggerakkan agen studio dengan skill khusus untuk memecahkan masalah dari fase *Discover*, *Define*, *Develop*, hingga *Deliver*.
6. **Workspace Riset & Synthesis**: Menghasilkan visualisasi *Empathy Map* dan *User Persona* secara otomatis dari teks masukan riset lapangan.
7. **Penyusun Proposal (Proposal Builder)**: Menyusun 20 bab prasyarat dokumen *2nd submission* yang dilengkapi indikator kriteria penilaian juri per bab.
8. **Simulasi Juri (Judge Simulation Mode)**: Alat bantu analisis naskah proposal berbasis kata kunci untuk memprediksi skor (1–5) per kriteria PIDI Hackathon, lengkap dengan analisis penalti keamanan dan rekomendasi perbaikan.
9. **Kanvas DT & Ekspor Pratinjau**: Halaman rangkuman visual 14 dimensi Kanvas Design Thinking JEDA yang siap diekspor ke format Markdown bersih untuk presentasi.

## 3. Roster Agen Studio (8 Pakar Interdisipliner)

Platform ini menggerakkan 8 agen AI virtual spesialis yang menyimulasikan kolaborasi perancangan produk:

1. **Raka (Desainer Utama - Pimpinan Studio)**: Fokus pada metodologi *Design Thinking*, penyelarasan masalah, dan kualitas interaksi pengguna.
2. **Dina (Spesialis Validasi & Peneliti Lapangan)**: Ahli dalam metodologi riset pengguna, pengumpulan data survei, dan wawancara empati.
3. **Fajar (Arsitek Produk & Keberlanjutan Sosial)**: Bertanggung jawab atas nilai publik, inklusi keuangan, dan keberlanjutan model bisnis kemasyarakatan.
4. **Bima (Insinyur Sistem Finansial & Keamanan)**: Mengamankan arsitektur sistem, memandu prinsip *Privacy-by-Design*, dan memetakan model optimalisasi data.
5. **Maya (Konsultan Kepatuhan Fintech & Regulasi)**: Mengawal kepatuhan terhadap regulasi OJK (POJK Perlindungan Konsumen) dan UU Pelindungan Data Pribadi (PDP No. 27/2022).
6. **Saras (Spesialis Calm UX & Pengalaman Inklusif)**: Mendesain antarmuka visual penenang stres kognitif dan interaksi mikro-narasi yang bersahabat.
7. **Arman (Analis Bisnis Sosial & Skalabilitas)**: Menganalisis model pendapatan subsidi silang B2B, program wellness korporat, dan dampak makroekonomi.
8. **Dr. Nadira (Konsultan Kesejahteraan Emosional & Psikiatri)**: Mengawal modul kesehatan mental, deteksi dini krisis berat, dan rujukan hotline krisis psikologi.

## 4. Cara Menjalankan Secara Lokal

### Prasyarat
* Node.js versi 18.x atau yang lebih baru
* npm (Node Package Manager)

### Langkah-langkah
1. Klon repositori ini atau navigasikan ke direktori proyek.
2. Pasang semua dependensi proyek:
   ```bash
   npm install
   ```
3. Jalankan server pengembangan lokal:
   ```bash
   npm run dev
   ```
4. Buka peramban (browser) dan akses alamat:
   ```
   http://localhost:3000
   ```

## 5. Batasan Hukum & Keselamatan (Safety Disclaimers)

* **Bukan Lembaga Keuangan**: JEDA bukan penyedia pinjaman, tidak menyalurkan kredit, dan tidak memfasilitasi pendanaan komersial dalam bentuk apa pun.
* **Bukan Broker Utang/Debt Collector**: JEDA tidak menjadi perantara mediasi komersial utang pengguna dan tidak terafiliasi dengan agen penagihan mana pun.
* **Bukan Penasihat Investasi**: JEDA tidak menjual produk investasi komersial (saham, kripto, reksa dana, obligasi) dan tidak merekomendasikan alokasi portofolio personal.
* **Bukan Nasihat Medis/Hukum**: Layanan asesmen krisis emosional JEDA tidak menggantikan diagnosis psikiatri klinis atau nasihat hukum bersertifikat.
* **Hotline Krisis**: Pengguna dengan indikasi stres mental berat/pikiran menyakiti diri segera diarahkan untuk menghubungi Hotline Kesehatan Jiwa Kemenkes 119 Ext. 8 atau psikolog klinis profesional terdekat.
* **Privacy-by-Design**: Platform dirancang untuk memproses data secara lokal di perangkat browser pengguna tanpa mengambil data pribadi sensitif seperti daftar kontak telepon, isi SMS, KTP, atau galeri foto.

---

## 6. Batasan Prototipe Saat Ini (Known Limitations)

1. **Client-Side Only**: Prototipe ini beroperasi sepenuhnya di sisi klien (*client-side*) menggunakan penyimpanan lokal (*localStorage*) peramban pengguna untuk menyimpan draf proposal. Tidak ada basis data server-side atau sistem otentikasi (auth) yang terpasang pada rilis demo ini.
2. **Data Survei Bersifat Mock**: Dashboard visual survei menggunakan data simulasi 450 responden sebagai demonstrasi model pembacaan insight sebelum dihubungkan dengan Google Form/CSV riil.
3. **Generator Lokal Deterministik**: Modul penalaran AI disimulasikan menggunakan generator deterministik lokal berbasis kata kunci untuk menjaga kecepatan respons dan keandalan demo tanpa memerlukan kunci API eksternal berbayar.

## 7. Rencana Pengembangan Masa Depan (Future Roadmap)

* **Integrasi API OJK & Kepolisian**: Menghubungkan modul *Debt Map* secara real-time ke database OJK untuk pengecekan legalitas instan serta portal patroli siber kepolisian untuk pelaporan pinjol ilegal langsung.
* **Enkripsi Kriptografi Lokal (WebCrypto API)**: Menerapkan skema enkripsi ujung-ke-ujung (*end-to-end local encryption*) penuh pada data pemetaan utang pengguna sehingga jaminan privasi data di perangkat terbukti secara matematis.
* **Modul Konseling Tatap Muka**: Bermitra dengan asosiasi psikologi klinis Indonesia untuk menyediakan jalur rujukan konseling video langsung (*tele-health*) terintegrasi bagi pengguna berisiko tinggi.
* **Portal HR B2B Kesejahteraan Karyawan**: Membangun portal analitik anonim bagi perwakilan HRD perusahaan mitra untuk melacak indeks stres keuangan kelompok karyawan (*employee financial wellness index*) guna membiayai operasional gratis bagi publik.
